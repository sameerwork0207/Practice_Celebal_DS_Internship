# Celebal Data Science Internship — Week 2

## Tesla Deliveries ML & Forecasting Pipeline

### Overview
This repository contains a clean, end-to-end Machine Learning and Time-Series pipeline to analyze and forecast Tesla's delivery data (2015–2025). 

The project satisfies all fundamental requirements of the Week 2 syllabus using a streamlined, easy-to-read approach.

### 🛠️ Pipeline Steps
1. **Data Preprocessing:** Standardizes column names and builds a chronological timeline.
2. **Exploratory Data Analysis (EDA):** Plots historical delivery trends.
3. **Feature Engineering:** Handles missing values and scales numeric data.
4. **Regression & Tuning:** Fits a Random Forest model and optimizes parameters using Grid Search.
5. **Time Series Forecasting:** Uses a SARIMAX model to project Tesla deliveries 5 years into the future (2026–2030).
